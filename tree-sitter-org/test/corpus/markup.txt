================
Invalid spaced markup markers stay plain text
================
- = not verbatim = (space before closing)
- * not bold * (space before closing)
- / not italic / (space before closing)
- _ not underline _ (space before closing)
- + not strike + (space before closing)
- ~ not code ~ (space before closing)

---

(document
  (zeroth_section
    (list_item

- "/italic inside double quotes/"
- '/italic inside single quotes/'
